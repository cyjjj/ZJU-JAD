# 多客户端聊天服务器

##由于未设计GUI，请用命令行运行server.jar和client.jar
先运行server.jar，使用端口8080，能同时接受多个客户端的连接
再运行client.jar，可同时打开多个客户端进入聊天室
java -jar server.jar
java -jar client.jar
……
java -jar client.jar

##登陆与注册
有十个初始账号，用户名和密码为"user1"和"pwd1"~"user10"和"pwd10"
也可以注册新账号，登陆时输入未注册过的账号和密码，自动注册
只有当账号存在且密码正确（或新注册）、该账号没有已登录，才可以成功登陆进入聊天室
客户端会显示对应通知信息
成功进入聊天室时所有客户端会收到通知信息

##群聊
将任意一个客户端发送的文本向所有客户端（包括发送方）转发
且客户端实时监控群聊，可以看到所有群聊消息以及通知信息

##退出聊天室
客户端输入"bye"即可退出聊天室，其他所有客户端和服务器收到对应通知信息
